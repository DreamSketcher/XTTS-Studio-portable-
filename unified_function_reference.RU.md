## 1. Конвейер обработки текста

### 1.1 TextNormalizer (`engine/normalizer.py`)

**Класс**: `TextNormalizer`

**Назначение**: Полноценная нормализация русского текста перед синтезом.

#### Основные методы

| Метод                    | Сигнатура                                      | Описание                              |
|--------------------------|------------------------------------------------|---------------------------------------|
| `normalize`              | `normalize(self, text: str) -> str`            | **Главная функция**                   |
| `safe_character_filter`  | `safe_character_filter(self, text: str) -> str`| Фильтрация символов (после WordReplacer) |

#### Внутренние методы
- `_yoficator()` — Ёфикация
- `_fix_abbrev_rhythm()`, `_fix_mixed_case_rhythm()`, `_fix_cyrillic_abbrev()`
- `_replace_time_and_ratio()`
- `_ordinal_neuter()`

---

### 1.2 WordReplacer (`engine/word_replacer.py`)

**Класс**: `WordReplacer`

**Назначение**: Система замены слов + автоматическая транслитерация.

#### Приоритет категорий

| Категория       | Приоритет | Источник                  |
|-----------------|-----------|---------------------------|
| `builtin`       | Низший    | Исторические              |
| `auto`          | Средний   | Автотранслит              |
| `ai_corrected`  | Высокий   | AI Conductor              |
| `custom`        | Высший    | Ручные правки             |

#### Основные методы

| Метод          | Сигнатура                                              | Описание                     |
|----------------|--------------------------------------------------------|------------------------------|
| `apply`        | `apply(self, text, persist_new=True)`                  | **Главная функция**          |
| `add_rule`     | `add_rule(self, word, replacement, category="custom")` | Добавление правила           |
| `remove_rule`  | `remove_rule(self, word)`                              | Удаление правила             |

---

### 1.3 TextChunker (`engine/chunker.py`)

**Класс**: `TextChunker`

**Параметры**: `max_size=175`, `target_size=150`, `min_size=50`

**Защитные правила**:
- Запрещено начинать чанк с: `и, а, но, или, который...`
- Запрещено заканчивать чанк на: `и, а, но, или, который...`

#### Основные методы

| Метод           | Сигнатура                        | Описание                     |
|-----------------|----------------------------------|------------------------------|
| `chunk_text`    | `chunk_text(self, text)`         | **Главная публичная функция**|
| `_split_sentences` | —                             | Безопасное разбиение         |
| `_split_long`   | —                                | Разбиение длинных предложений|
| `_merge`        | —                                | Склеивание коротких чанков   |

---

### 1.4 SmartPauseEngine (`engine/smart_pauses.py`)

**Класс**: `SmartPauseEngine`

#### Основные методы

| Метод             | Сигнатура                                      | Описание                     |
|-------------------|------------------------------------------------|------------------------------|
| `get_pause_ms`    | `get_pause_ms(self, chunk, next_chunk="")`     | **Главная функция**          |
| `detect_emotion`  | `detect_emotion(self, chunk)`                  | Определение эмоции           |

> **Важно**: При активном AI Conductor этот модуль отключается.

## 2. Интеграция AI и LLM

### 2.1 GPT Client (`engine/gpt_client.py`)

**Назначение**: Универсальный клиент для OpenAI-совместимых провайдеров.

#### Основные функции

| Функция                | Описание                              |
|------------------------|---------------------------------------|
| `get_provider()` / `set_provider()` | Управление провайдером     |
| `chat()`               | Свободный чат                         |
| `improve_for_tts()`    | Улучшение текста для TTS              |
| `_call_with_chain()`   | **Главная точка входа** (цепочка fallback) |

#### Поддерживаемые провайдеры
`groq`, `openrouter`, `proxy`, `local`, кастомные

---

### 2.2 Local LLM Client (`engine/local_llm_client.py`)

**Назначение**: Инференс GGUF-моделей через `llama-cpp-python`.

#### Основные функции

| Функция                    | Описание                              |
|----------------------------|---------------------------------------|
| `call_local_llm()`         | **Главная функция генерации**         |
| `download_model()`         | Скачивание с resume                   |
| `register_model()`         | Регистрация модели                    |
| `_get_llm()`               | Ленивая загрузка модели               |

---

### 2.3 AI Conductor (`engine/ai_conductor.py`)

**Главная функция**:

```python
def conduct(text, chunks, rewrite_enabled=False, ...) -> list | dict | None
```

#### Возвращаемые параметры

| Параметр              | Диапазон     | Назначение                    |
|-----------------------|--------------|-------------------------------|
| `temperature`         | 0.50–0.90    | Вариативность интонации       |
| `speed`               | 0.75–1.25    | Скорость речи                 |
| `pause_after_ms`      | 0–1200       | Длительность паузы            |
