## Принятое исключение по безопасности

### CVE-2025-3000 — PyTorch `torch.jit.script`

- **Пакет:** `torch==2.11.0`
- **Исправленная версия:** отсутствует в базе известных уязвимостей на момент составления отчёта.
- **Причина использования этой версии:** семейство `torch==2.11.0` / `torchaudio==2.11.0` / `torchvision==0.26.0` является взаимно протестированной сборкой под Windows с CUDA-поддержкой. Повышение версии только одного компонента нарушает бинарную ABI-совместимость с `torchaudio`, используемым в движке XTTS.
- **Риски и ограничения:** уязвимость затрагивает обработку скриптов `torch.jit.script`. XTTS Studio не предоставляет внешнего API, принимающего произвольный код на Python или TorchScript.
- **Компенсирующие меры:** сторонние модели RVC `.pth` требуют ясного подтверждения доверия, привязанного к SHA-256; эмбеддинги загружаются с параметром `weights_only=True`; скачивание моделей из сети не исполняет произвольный код.
- **Срок пересмотра:** до выходя совместимого обновления PyTorch с исправлением данной уязвимости.
