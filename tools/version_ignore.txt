# Чёрный список для version.json (синтаксис как у .gitignore)

# - Пустые строки и строки с # игнорируются.
# - Строка, оканчивающаяся на "/"  — исключает целую папку (и всё внутри неё).
# - Строка с "*", "?", "[...]"     — работает как маска (fnmatch), например *.PNG
# - Обычная строка                — точное имя файла ИЛИ путь (совпадение по
#                                    полному относительному пути или по имени файла).
#
# Просто добавляй сюда новые строки, когда понадобится исключить ещё что-то —
# ничего в коде трогать не нужно, generate_version_files.py читает этот файл
# при каждом запуске.
